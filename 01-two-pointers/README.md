
## Two Pointers Pattern

Used when :
- Question is on Arrays/Linked List.
- Sorted data/Easy if sorted.
- Keywords: Merge/Remove duplicates/Rearrange/Find pairs,triplets,etc.
- If Linked List->Detect Cycle.
- If to find data->sort array.
- If to find index->array should be sorted.

### Problems Solved
- Pair with Target Sum (easy)
- Rearrange 0 and 1
- Remove Duplicates (easy)
- Squaring a Sorted Array (easy)
- Triplet Sum to Zero (medium)
- Triplet Sum Close to Target (medium)
- Triplets with Smaller Sum (medium)
- Subarrays with Product Less than a Target (medium)
- Dutch National Flag Problem (medium)
- Quadruple Sum to Target (medium)
- Comparing Strings containing Backspaces (medium)
- Minimum Window Sort (medium)
